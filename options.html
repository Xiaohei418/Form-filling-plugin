同一天的多条记录会合并到同一行。第二列分钟数会自动重算。

如何获取 Web App 地址

  1. 打开你要写入的 Google Sheet。
  2. 菜单 扩展程序 → Apps Script。
  3. 清空默认代码，粘贴项目里 Code.gs 的内容，保存。
  4. Deploy → New deployment，齿轮选 Web app。
  5. "Execute as" 选 Me；"Who has access" 选 Anyone。
  6. 授权后复制 /exec 结尾的 URL。

改了 Code.gs 后必须 Manage deployments → ✏️ → New version 重新部署才生效。
